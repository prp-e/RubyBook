##عملگرهای محاسباتی

عملگر جمع : +

مثال :
```
a = 2
b = 3
c = a + b
```

عملگر تفریق : -

مثال:
```
a = 2
b = 3
c = a - b
```

عملگر ضرب : *

مثال :
```
a = 2
b = 3
c = a * b
```

عملگر تقسیم : /

مثال :
```
a = 2
b = 3
c = a /b
```

عملگر باقیمانده تقسیم : %

مثال :
```
a = 2
b = 3
c = a % b
```

عملگر توان : ^^

مثال :
```
a = 2
b = 3
c = a ^^ b
```
